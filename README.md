# Linear Regression Project

## Objective
To implement and understand simple and multiple linear regression.

## Dataset
House Price Prediction Dataset

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Steps Performed
1. Loaded dataset
2. Converted categorical data into numerical values
3. Selected multiple features
4. Split data into training and testing sets
5. Trained Linear Regression model
6. Evaluated using MAE, MSE, and R2 Score
7. Visualized results using scatter plot

## Results
The model predicts house prices based on multiple features.
The scatter plot shows the relationship between area and price.
